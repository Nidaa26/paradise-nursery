Paradise Nursery is a fully functional React-based e-commerce shopping application for an online plant shop. This project was built as the final project for the IBM Full Stack Developer course and demonstrates core React and Redux skills in a real-world scenario.

🌱 Project Overview
Paradise Nursery allows users to:

Browse a curated collection of houseplants organized by category
View plant details including name, image, description, and price
Add plants to a shopping cart
Manage cart items (increase/decrease quantity, remove items)
View a dynamically updating cart count in the navbar
